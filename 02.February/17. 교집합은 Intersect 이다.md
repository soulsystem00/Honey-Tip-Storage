# 교집합은 Intersect 이다.  

교집합만 단독으로 작성하는게 맞나 싶긴 한데  
일단 자주 쓰는 것 같으니깐  
교집합은 Intersect임  
</br>
배열 A 가 있고 배열 B가 있을 때  
두 배열의 교집합을 구하고 싶으면 Intersect를 사용하면 됨  
</br>
헷갈리지 말아야 할게  
Array 클래스에서 사용하는게 아니라  
그냥 배열에서 사용하면 되고  
Linq를 임포트 시켜야 하고  
배열 A에서 함수를 사용하면 됨  
</br>
교집합 같은 경우에는 굳이 배열 A에서 사용하지 않아도 되지만  
교환법칙이 성립하지 않는 경우도 있어서  
기억을 해놓는게 좋음  
</br>
```
arrayA.Intersect(arrayB).ToArray();
```
